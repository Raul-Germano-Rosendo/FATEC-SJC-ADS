## Sobre este Repositório

Este repositório contém os projetos e materiais desenvolvidos no curso de **Análise e Desenvolvimento de Sistemas (ADS)** da FATEC São José dos Campos.

### Conteúdo

- Projetos acadêmicos
- Exercícios práticos
- Trabalhos em equipe
- Atividades de disciplinas

### Objetivo

Documentar e organizar o aprendizado ao longo do curso, compartilhando código e conhecimento adquirido nas disciplinas de programação, banco de dados, engenharia de software e outras áreas.

### Tecnologias

Diversos projetos utilizam diferentes linguagens e frameworks, como:
- Java, Python, JavaScript
- Banco de dados (SQL, MongoDB)
- Frameworks web e mobile
